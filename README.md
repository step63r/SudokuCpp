# SudokuCpp

## Description

Solve Sudoku (number place) with DFS (backtracking).

## Requirement

- Windows 10 Pro (64-bit)
- Visual Studio 2019

## Usage

Build this with Visual Studio 2019 and run.

## Install

Fork and clone this repository.

```
$ git clone git@github.com:yourname/SudokuApp.git
```

## For Application Users

Input as below,

```
53**7****
6**195***
*98****6*
8***6***3
4**8*3**1
7***2***6
*6****28*
***419**5
****8**79
```

and be solved as below.

```
5 3 4 6 7 8 9 1 2
6 7 2 1 9 5 3 4 8
1 9 8 3 4 2 5 6 7
8 5 9 7 6 1 4 2 3
4 2 6 8 5 3 7 9 1
7 1 3 9 2 4 8 5 6
9 6 1 5 3 7 2 8 4
2 8 7 4 1 9 6 3 5
3 4 5 2 8 6 1 7 9
```
