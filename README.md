# SudokuCpp

## Description

Solve Sudoku (number place) with DFS (backtracking).

## Requirement

- Windows 10 Pro (64-bit)
- Visual Studio 2019

## Usage

Build this with Visual Studio 2019 and run.

## Install

Fork and clone this repository.

```
$ git clone git@github.com:yourname/SudokuApp.git
```
